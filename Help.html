Frequently Asked Questions

How do I navigate between different sections?

You can navigate between sections using the sidebar tabs on the left. Each tab represents a different section like My Feed, Kudos, People, Revenue, My Work, and My Plan. You can also use the Dashboard view to see all sections at once.

How do I add a new Kudos entry?

In the Kudos section, click the "Give Kudos" button. This will open a modal where you can enter the recipient's name and your message. Click "Submit" to save the kudos entry.

How do I create a new plan?

In the My Plan section, you can add both Weekly and Quarterly plans. Click the "Add Weekly Plan" or "Add Quarterly Plan" button, fill in the required information, and submit. The system will automatically validate for duplicate entries.

What is the difference between Home and Dashboard views?

The Home view shows one section at a time with auto-rotating tabs, while the Dashboard view displays all sections in a grid layout without auto-rotation. Both views share the same data, so changes in one will reflect in the other.

How do I access the chatbot?

The chatbot is accessible via the chat icon in the bottom-right corner of the screen. Click on it to open the chat interface. You can minimize or close the chat window using the buttons in the chat header.

How do I edit or delete entries?

Each entry has action icons (edit, view, delete) on the right side. Click the edit icon to modify an entry, the view icon to see details, or the delete icon to remove an entry. A confirmation dialog will appear for deletions.
